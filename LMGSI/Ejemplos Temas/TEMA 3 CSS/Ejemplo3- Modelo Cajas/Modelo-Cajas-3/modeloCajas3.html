El texto en el elemento div 5º flota alrededor de los otros div y el elemento div 5º rodea todo. ¿Qué pasaría si quisiéramos que el elemento div5º apareciera despues de los elemento que están flotando?. Pues para despejar elementos que están flotando a la izquierda debes usar la propiedad clear con el valor left. Para despejar elementos que flotan a la derecha usa clear:right; y para despejar a ambos lados clear:both;
